# Nexus Laravel Notifier Adapter

Laravel adapter package for Nexus Notifier integration.

## Package

- Composer: `azaharizaman/nexus-laravel-notifier-adapter`
- Namespace: `Nexus\Laravel\Notifier\`

## Installation

```bash
composer require azaharizaman/nexus-laravel-notifier-adapter
```

## Purpose

This package provides Laravel-specific notifier integrations, including queue-backed email delivery adapters.
